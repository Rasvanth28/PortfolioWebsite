# 🦋 Terminal Portfolio: Cicada-3301 Edition

An interactive, cryptography-themed developer portfolio built for **K.C. Rasvanth**. Designed to break away from traditional web layouts, this portfolio uses a simulated terminal interface, an ethereal particle system, and dark-space aesthetics to create an immersive "hacker" experience.

---

## ✨ Core Features

*   **💻 Interactive Terminal Console**: A fully functional command-line interface. Users can navigate the site via commands like `whois`, `skills`, `projects`, `contact`, and `clear`.
*   **🦋 The Butterfly Effect**: A high-performance HTML5 Canvas background rendering a flock of glowing butterflies. Interaction triggers "Reality Anomalies" with global color-shifts and particle explosions.
*   **📧 Terminal-Integrated Contact**: No traditional forms. Secure transmissions are sent via the `sendMail` command, integrated with Formspree for backend processing.
*   **🛡️ Hidden Reveal Protocol**: A secret `reveal` command that triggers a high-definition local video overlay, replacing the standard YouTube embed for a more seamless experience.
*   **🎨 Premium Aesthetics**: A minimalist `#050505` palette paired with **Deep Violet** (`#9d4edd`) neon accents, frosted glassmorphism, and **JetBrains Mono** typography.

## 🛠 Technical Stack

| Category | technologies |
| :--- | :--- |
| **Frontend** | HTML5 (Semantic), CSS3 (Custom Variables, Glassmorphism), Vanilla JavaScript |
| **Icons** | [DevIcons](https://devicon.dev/) (Integrated via CDN for peak performance) |
| **Particles** | HTML5 Canvas API (Object-Oriented Animation) |
| **Backend** | Formspree API for secure mail transmission |
| **Media** | Native HTML5 Video for the "Reveal" sequence |

## 🚀 Deployment & Local Hosting

### Method 1: Python (Quickest)
```bash
python3 -m http.server 8000
```
Then visit `http://localhost:8000`.

### Method 2: Live Server (VS Code)
Right-click `index.html` and select **"Open with Live Server"**.

### Method 3: GitHub Pages
The project is optimized for GitHub Pages. Ensure `reveal.mp4` is present in the root directory for the full experience.
